Box Hunt

Score: 0 Time left: 60s

Instructions

This game is similar to the retro 2D version of Whac-A-Mole. When the coloured box appears, try to hit it. The more hits you achieve in 60 seconds, the more points you score. But beware, the higher your score, the faster the box moves!


  • Press [ Start ]

  • Hit the coloured box

  • Game stops after 60 seconds

  • Press [ Reset ] to clear the score

  • Press [ Start ] to play again
